TensorFlow 初试

安装

  1. pip install tensorflow

  2. 如果安装GPU版本的，需要安装 Cuda Toolkit 8.0 和 cuDNN v5.1
    pip install tensorflow-gpu
    并且设置环境变量 LD_LIBRARY_PATH和CUDA_HOME。

  3. 运行 demo model

    python -m tensorflow.models.image.mnist.convolutional
